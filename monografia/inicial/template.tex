\resumo
A Web Semântica é uma visão proposta por Tim Berners-Lee em que os documentos da Web vão possuir significado semântico e que as máquinas serão capazes de tomar decisões baseadas em regras. Entre as várias tecnologias relacionadas à Web Semântica, estão os formalismos de representação de conhecimento e os algoritmos para prova de teoremas.

Este trabalho propôe implementar parte do algoritmo do método das conexões para lógica de descrição, usado para fazer atividades de raciocínio em uma base de conhecimento.  O módulo a ser implementado é a conversão dos axiomas da base de conhecimento para a forma normal positiva.
\begin{keywords}
método das conexões, lógica de descrição, lógica, inteligencia artificial
\end{keywords}

\abstract
The Semantic Web is a dream of Tim Berners-Lee in which Web documents will have semantic meaning and that the machines will be able to make decisions based on rules. Among the various technologies related to Semantic Web are the formalisms for knowledge representation and algorithms for theorem proving.

This paper proposes to implement part of the algorithm of the connections method for description logic, used to do activities of reasoning in a knowledge base. The module to be implemented is the conversion of the axioms of the knowledge base for the positive normal form.


\begin{keywords}
connection method, description logic, artificial intelligence
\end{keywords}

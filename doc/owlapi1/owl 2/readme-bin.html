An API for OWL

This is the binary distribution of the OWL API code. Libraries can be found in the lib directory, documentation in the javadoc directory.

Sample Application

A simple sample application is included. This will read an XML/RDF file from a URL and either perform validation, or produce some concrete rendering of the ontology. Try:

processor -?

for details.

Package Specification

This package provides a simple implementation of a {@link org.semanticweb.owl.io.Renderer Renderer} that renders an ontology according to the OWL Abstract syntax. This is primarily intended as an assistant to debugging and simple user interfaces, rather than as a primary mechanism for storing ontologies.
